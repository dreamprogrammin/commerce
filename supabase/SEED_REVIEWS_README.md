# Скрипт для добавления тестовых отзывов

## Использование

### Локальная база данных (Supabase Local)

```bash
# Запустите Supabase локально
supabase start

# Выполните скрипт
psql postgresql://postgres:postgres@localhost:54322/postgres -f supabase/seed_reviews.sql
```

### Через Supabase Studio

1. Откройте http://localhost:54323 (Supabase Studio)
2. Перейдите в SQL Editor
3. Скопируйте содержимое файла `supabase/seed_reviews.sql`
4. Нажмите "Run"

### Через продакшн базу данных

```bash
# Получите connection string из Supabase Dashboard
psql "postgresql://postgres:[YOUR-PASSWORD]@[YOUR-PROJECT-REF].supabase.co:5432/postgres" -f supabase/seed_reviews.sql
```

## Что делает скрипт

1. **Создает 5 тестовых пользователей:**
   - testuser1@example.com
   - testuser2@example.com
   - testuser3@example.com
   - testuser4@example.com
   - testuser5@example.com

2. **Добавляет по 3 отзыва для первых 5 активных товаров:**
   - Рейтинги: от 3 до 5 звезд
   - Тексты: реалистичные отзывы на русском языке
   - Даты: случайные за последние 30 дней

3. **Автоматически обновляет рейтинги товаров:**
   - Триггер `update_product_review_stats` пересчитывает `avg_rating` и `review_count`

4. **Показывает статистику:**
   - Список товаров с рейтингами
   - Общая статистика по отзывам
   - Распределение оценок (гистограмма)

## Результат

После выполнения скрипта:

- ✅ ProductCard отображает звезды и рейтинг
- ✅ Показывается количество отзывов
- ✅ RPC функция `get_filtered_products()` возвращает `avg_rating` и `review_count`

## Очистка тестовых данных

Если нужно удалить тестовые отзывы:

```sql
-- Удалить отзывы от тестовых пользователей
DELETE FROM product_reviews
WHERE user_id IN (
  '11111111-1111-1111-1111-111111111111',
  '22222222-2222-2222-2222-222222222222',
  '33333333-3333-3333-3333-333333333333',
  '44444444-4444-4444-4444-444444444444',
  '55555555-5555-5555-5555-555555555555'
);

-- Удалить тестовых пользователей
DELETE FROM auth.users
WHERE id IN (
  '11111111-1111-1111-1111-111111111111',
  '22222222-2222-2222-2222-222222222222',
  '33333333-3333-3333-3333-333333333333',
  '44444444-4444-4444-4444-444444444444',
  '55555555-5555-5555-5555-555555555555'
);
```

## Примечания

- Один пользователь может оставить только один отзыв на товар (constraint `product_reviews_user_id_product_id_key`)
- Отзывы автоматически обновляют рейтинг товара через триггер
- Неопубликованные отзывы (`is_published = false`) не учитываются в рейтинге
