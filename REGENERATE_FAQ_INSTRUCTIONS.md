# 🚀 Инструкция: Регенерация FAQ для Google PAA

## Что нужно сделать

Запустить массовую регенерацию FAQ для всех категорий, чтобы применить новую PAA-оптимизацию.

## Шаги

### 1. Открыть Supabase Studio

Перейти по ссылке: https://supabase.com/dashboard/project/YOUR_PROJECT_ID/editor

### 2. Открыть SQL Editor

- В левом меню выбрать **SQL Editor**
- Нажать **New query**

### 3. Скопировать и запустить скрипт

Открыть файл `supabase/regenerate_faq_for_paa.sql` и скопировать весь его содержимый в SQL Editor.

Или просто выполнить эту команду:

```sql
SELECT * FROM public.generate_questions_for_all_categories();
```

### 4. Нажать "Run" (или Ctrl+Enter)

Скрипт выполнится за 5-10 секунд.

### 5. Проверить результат

После выполнения ты увидишь таблицу с результатами:

```
category_id                          | questions_count | is_premium
-------------------------------------|-----------------|------------
uuid-категории-1                     | 4               | true
uuid-категории-2                     | 4               | false
...
```

### 6. Проверить на сайте

1. Открыть любую страницу категории: https://uhti.kz/catalog/boys/mashinki/radioupravlyaemye-mashinki
2. Прокрутить до блока "Часто задаваемые вопросы"
3. Убедиться, что ответы теперь содержат:
   - ✅ Списки (маркированные)
   - ✅ Жирный текст в ключевых местах
   - ✅ Ссылки на каталог
   - ✅ Четкие определения в первом предложении

### 7. Проверить Schema.org

Открыть Google Rich Results Test:

```
https://search.google.com/test/rich-results?url=https://uhti.kz/catalog/boys/mashinki/radioupravlyaemye-mashinki
```

Убедиться, что:

- ✅ FAQPage обнаружена
- ✅ Все вопросы имеют чистый текст (без HTML-тегов)
- ✅ Нет ошибок валидации

## Что изменится

### До регенерации:

```
Вопрос: В каком ценовом диапазоне товары в категории "Радиоуправляемые машинки"?
Ответ: Цены на товары в категории "Радиоуправляемые машинки" варьируются от 12 990 ₸ до 41 000 ₸.
Вы можете найти как бюджетные варианты, так и премиум товары.
```

### После регенерации:

```
Вопрос: Сколько стоят Радиоуправляемые машинки в Алматы?
Ответ: Цены на Радиоуправляемые машинки в Алматы варьируются от 12 990 ₸ до 41 000 ₸.
У нас вы найдете:
• Бюджетные варианты для экономных покупателей
• Премиум товары известных брендов
• Акции и скидки на популярные модели
Смотреть все цены на Радиоуправляемые машинки.
```

## Если что-то пошло не так

Проверить логи в Supabase:

1. Перейти в **Logs** → **Postgres Logs**
2. Найти ошибки, связанные с `generate_category_questions`
3. Написать в чат с описанием ошибки
