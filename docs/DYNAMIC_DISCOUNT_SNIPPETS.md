# 🔥 Dynamic Discount Snippets - Документация

> **Дата реализации:** 31 марта 2026  
> **Цель:** Показывать реальные скидки в Google PAA вместо статичного текста

---

## 📋 Проблема

### До оптимизации:

```sql
SELECT MIN(price), MAX(price) FROM products
```

**Что было не так:**

1. ❌ Функция смотрела только на `price`, игнорируя `discount_percentage`
2. ❌ Статично писали "Акции и скидки", даже если их нет
3. ❌ Упускали возможность показать реальные скидки -50% в Google

**Пример старого ответа:**

```
Вопрос: Сколько стоят Радиоуправляемые машинки в Алматы?
Ответ: Цены варьируются от 12 990 ₸ до 41 000 ₸.
У нас вы найдете:
• Бюджетные варианты
• Премиум товары
• Акции и скидки на популярные модели  ← статичный текст
```

---

## ✅ Решение

### После оптимизации:

```sql
-- Реальная минимальная цена с учетом скидки
MIN(
  CASE
    WHEN p.discount_percentage > 0 THEN
      ROUND(p.price * (100 - p.discount_percentage) / 100)
    ELSE
      p.price
  END
)

-- Максимальная скидка в категории
MAX(COALESCE(p.discount_percentage, 0))
```

**Что изменилось:**

1. ✅ Высчитываем реальную `min_price` с учетом `discount_percentage`
2. ✅ Находим максимальную скидку (`max_discount`) в категории
3. ✅ Динамически добавляем "🔥 Прямо сейчас действуют скидки до X%!" если есть

**Пример нового ответа (если есть скидки):**

```
Вопрос: Сколько стоят Радиоуправляемые машинки в Алматы?
Ответ: Цены варьируются от 9 990 ₸ до 41 000 ₸.
🔥 Прямо сейчас действуют скидки до 30%!  ← динамический текст
У нас вы найдете:
• Бюджетные варианты
• Премиум товары
• Товары со скидками до 30% на популярные модели  ← динамический текст
```

**Пример нового ответа (если скидок нет):**

```
Вопрос: Сколько стоят Конструкторы в Алматы?
Ответ: Цены варьируются от 2 990 ₸ до 85 000 ₸.
У нас вы найдете:
• Бюджетные варианты
• Премиум товары
• Акции и скидки на популярные модели  ← статичный текст (как раньше)
```

## 🎯 Что изменилось в коде

### 1. Добавлена переменная `v_max_discount`

```sql
DECLARE
  v_max_discount INTEGER;  -- 🔥 Новая переменная
```

### 2. Обновлен SELECT с расчетом реальных цен

```sql
SELECT
  -- ... другие поля ...

  -- 🔥 Реальная минимальная цена с учетом скидки
  MIN(
    CASE
      WHEN p.discount_percentage > 0 THEN
        ROUND(p.price * (100 - p.discount_percentage) / 100)
      ELSE
        p.price
    END
  ),

  -- 🔥 Реальная максимальная цена с учетом скидки
  MAX(
    CASE
      WHEN p.discount_percentage > 0 THEN
        ROUND(p.price * (100 - p.discount_percentage) / 100)
      ELSE
        p.price
    END
  ),

  -- 🔥 Максимальная скидка в категории
  MAX(COALESCE(p.discount_percentage, 0))
INTO
  v_min_price,
  v_max_price,
  v_max_discount  -- 🔥 Новая переменная
```

### 3. Динамическая вставка скидки в ответ

```sql
'<strong>Цены на ' || v_category_name || ' в Алматы варьируются от ' ||
replace(to_char(v_min_price::bigint, 'FM999,999,999'), ',', ' ') || ' ₸ до ' ||
replace(to_char(v_max_price::bigint, 'FM999,999,999'), ',', ' ') || ' ₸</strong>. ' ||

-- 🔥 Динамическая вставка скидки
CASE
  WHEN v_max_discount > 0 THEN
    '🔥 <strong>Прямо сейчас действуют скидки до ' || v_max_discount || '%!</strong> '
  ELSE ''
END ||

'У нас вы найдете:<ul>' ||
'<li>Бюджетные варианты для экономных покупателей</li>' ||
'<li>Премиум товары известных брендов</li>' ||

-- 🔥 Динамический текст в зависимости от наличия скидок
CASE
  WHEN v_max_discount > 0 THEN
    '<li>Товары со скидками до ' || v_max_discount || '% на популярные модели</li>'
  ELSE
    '<li>Акции и скидки на популярные модели</li>'
END ||
'</ul>'
```

## 🚀 Как применить

### 1. Применить миграцию

```sql
-- Выполнить в Supabase Studio → SQL Editor
-- Файл: supabase/migrations/20260331095500_add_dynamic_discount_snippets.sql
```

Или скопировать содержимое миграции и выполнить в SQL Editor.

### 2. Регенерировать FAQ для всех категорий

```sql
SELECT * FROM public.generate_questions_for_all_categories();
```

### 3. Проверить результат

```sql
-- Найти категории со скидками
SELECT
  c.name,
  MAX(p.discount_percentage) as max_discount,
  COUNT(*) as products_with_discount
FROM categories c
JOIN products p ON p.category_id = c.id
WHERE p.discount_percentage > 0 AND p.is_active = true
GROUP BY c.id, c.name
ORDER BY max_discount DESC
LIMIT 10;

-- Проверить ответ на вопрос о ценах
SELECT
  c.name,
  cq.question_text,
  LEFT(cq.answer_text, 300) as answer_preview
FROM category_questions cq
JOIN categories c ON c.id = cq.category_id
WHERE cq.question_text LIKE 'Сколько стоят%'
  AND cq.is_auto_generated = true
ORDER BY c.name
LIMIT 5;
```
