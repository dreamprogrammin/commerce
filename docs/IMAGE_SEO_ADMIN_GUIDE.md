# ✅ Генерация Alt-текстов через админку

## 🚀 Самый простой способ (30 секунд)

### Шаг 1: Открой админку
```
https://your-site.com/admin/image-seo
```

### Шаг 2: Нажми кнопку
```
[🔄 Сгенерировать Alt-тексты]
```

### Шаг 3: Дождись завершения
Увидишь статистику:
- ✅ Обновлено: X изображений
- 📷 Пропущено: Y изображений (уже есть качественные alt-тексты)
- 📊 Всего: Z изображений

## Готово! 🎉

Alt-тексты обновлены в базе данных и сразу работают на сайте.

---

## Альтернативные способы

### Через npm скрипт (локально)
```bash
npm run seo:generate
```

### Через API
```bash
curl -X POST https://your-site.com/api/generate-alt-texts
```

---

## Когда запускать?

- ✅ После добавления новых товаров
- ✅ После массовой загрузки изображений
- ✅ Периодически (раз в неделю/месяц)

---

## Что генерируется?

**Формат:** `[Бренд] [Название] [Серия] [Контекст]`

**Примеры:**
- `LEGO Конструктор Железный Человек Marvel купить в Казахстане`
- `Barbie Кукла Принцесса Disney вид сбоку`
- `Hot Wheels Трек Мертвая петля детальное фото`
