## 1. Исправление уведомлений о гостевых заказах

**Проблема**: Гостевые заказы не отправляли уведомления в Telegram админу.

**Причина**: Функция `create_guest_checkout` создавала заказы в таблице `orders` вместо `guest_checkouts`, из-за чего триггер `trigger_notify_guest_checkout` не срабатывал.

**Решение**:
- Исправлена функция `create_guest_checkout` для создания заказов в `guest_checkouts`
- Добавлены колонки: `source`, `delivery_cost`, `telegram_message_id`, `comment`
- Создан триггер для отправки уведомлений
- Удален дублирующий триггер `guest_checkouts_telegram_notification_v2`

**Миграции**:
- `20260414000001_fix_guest_checkout_telegram_notifications.sql`
- `20260414000002_ensure_guest_checkout_trigger.sql`
- `20260414000003_remove_duplicate_guest_trigger.sql`

### 3.2. Интеграция с Magic Links

**Проблема**: Уведомления не использовали существующий пайплайн с автологином.

**Решение**:
- Переработана функция `check_abandoned_carts` для использования пайплайна через `notifications`
- Триггер `trigger_telegram_on_notification` автоматически генерирует magic link
- Добавлена кнопка "🔗 Открыть на сайте" с автологином

**Миграция**: `20260414000010_use_notifications_pipeline.sql`

### 3.3. Система "Правила тишины" и промокодов

**Реализовано**:

**Таблица `reminder_logs`**:
- История всех отправленных напоминаний
- Используется для "правила тишины" (не спамить)
- Хранит snapshot корзины на момент отправки

**Двухэтапная система напоминаний**:

1. **Первое напоминание (20 минут)**:
   - Мягкое напоминание с фото товара
   - Персонализированный текст с названием
   - Кнопка с magic link для автологина
   - Условия: корзина не обновлялась 20+ минут, нет заказа, не было отправлено ранее

2. **Второе напоминание (24 часа)**:
   - Срочность: "🔥 Не забудьте про корзину!"
   - Персональный промокод `TG5-XXXXX` (5% скидка, 2 часа)
   - Условия: прошло 24+ часа, было первое напоминание, пользователь не купил

**Автоматизация**:
- Cron каждые 15 минут: `SELECT public.check_abandoned_carts()`
- Индексы для производительности при 10,000+ пользователей

**Миграция**: `20260414000011_smart_abandoned_cart_with_promo.sql`
