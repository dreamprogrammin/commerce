# Быстрый старт: Защита SEO-текстов

## Что изменилось?

Кнопка **"🤖 Автогенерация"** в `/admin/brand-seo` теперь **НЕ перезаписывает** уникальные SEO-тексты, написанные вручную.

## Как это работает?

Текст защищён от перезаписи, если:
- **Длина > 300 символов** ИЛИ
- **Содержит HTML-теги** (`<h1>`, `<div>`, `<ul>`, `<strong>`, и т.д.)

## Инструкция для контент-менеджера

### 1. Напишите уникальный текст

Откройте `/admin/brand-seo` и создайте/отредактируйте запись:

```html
<h2>Конструкторы LEGO для мальчиков в Алматы</h2>
<p>Ухтышка предлагает широкий выбор оригинальных конструкторов LEGO...</p>
<ul>
  <li>Официальная гарантия</li>
  <li>Доставка день-в-день</li>
</ul>
```

### 2. Сохраните запись

Нажмите **"Сохранить"**.

### 3. Запустите автогенерацию

Теперь можно безопасно нажать **"🤖 Автогенерация"** — ваш текст **НЕ будет перезаписан**.

## Что обновляется?

При автогенерации:
- ✅ **Обновляются**: `seo_h1`, `seo_title`, `seo_description`
- ✅ **Обновляется**: `seo_text` для коротких шаблонных текстов
- ❌ **НЕ обновляется**: `seo_text` для уникальных текстов (длинных или с HTML)

## Кнопки в админке

| Кнопка | Действие | Безопасность |
|--------|----------|--------------|
| **🤖 Автогенерация** | Создаёт новые записи, обновляет шаблонные тексты | ✅ Безопасно |
| **🔄 Перезаписать все** | Перезаписывает ВСЕ тексты, включая уникальные | ⚠️ Опасно |

## Проверка

После сохранения уникального текста проверьте в консоли браузера:

```javascript
// Запустите автогенерацию и проверьте логи
// Должно быть: "Protected: Категория + Бренд"
```
