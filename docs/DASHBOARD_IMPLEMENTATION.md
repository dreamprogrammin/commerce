# Sales Dashboard (Дашборд аналитики)

## Что было сделано

### 1. SQL RPC функция

**Миграция**: `supabase/migrations/20260317000002_create_dashboard_stats_rpc.sql`

Функция `get_dashboard_stats()` возвращает JSON с:

| Поле | Описание |
|------|----------|
| `revenue_month` | Выручка за текущий месяц (confirmed + delivered) |
| `revenue_total` | Выручка за всё время |
| `orders_month` | Заказы за текущий месяц (без отменённых) |
| `orders_total` | Заказы за всё время (без отменённых) |
| `orders_pending` | Новые заказы (new/pending — ожидают обработки) |
| `orders_delivered` | Доставленные заказы (всего) |
| `avg_order_month` | Средний чек за текущий месяц |
| `top_products` | ТОП-5 товаров по `sales_count` (с картинкой) |
| `revenue_by_day` | Выручка за последние 7 дней (по дням) |

Учитывает **обе таблицы**: `orders` (авторизованные) и `guest_checkouts` (гостевые).

### 2. Админ-дашборд

**Файл**: `pages/admin/index.vue`

Полностью заменена старая страница-заглушка на полноценный дашборд:

**Карточки статистики** (grid 2x2 на мобильных, 4 в ряд на десктопе):
- Выручка за месяц (+ подпись "всего за всё время")
- Заказы за месяц (+ подпись "всего")
- Новые заказы (оранжевый цвет если > 0)
- Средний чек за месяц

**Графики** (grid 1x2 на десктопе):
- **Топ-5 товаров** — прогресс-бары (shadcn `<Progress>`) с картинкой, названием и количеством продаж
- **Выручка за 7 дней** — столбчатый график на Tailwind (без внешних библиотек), дни недели + даты

**Быстрые ссылки** (grid 4 карточки):
- Товары, Категории, Бренды, Поставщики

**UX**:
- Skeleton-загрузка для всех блоков
- Кнопка "Обновить" с анимацией спиннера
- Адаптивная верстка для всех размеров экрана

### Использованные shadcn компоненты

- `Card`, `CardHeader`, `CardTitle`, `CardContent`
- `Badge`
- `Progress`
- `Skeleton`
- `Button`

## Как применить

```bash
# Применить миграцию
supabase db push

# Или локально
supabase db reset
```
