## Что нужно сделать

### Шаг 1: Проверь текущее состояние

```bash
npx tsx scripts/audit-alt-texts.ts
```

Это покажет:
- Сколько изображений без alt-текстов
- Сколько с дефолтными alt-текстами
- Примеры хороших и плохих alt-текстов

### Шаг 2: Обнови alt-тексты

```bash
npx tsx scripts/generate-alt-texts.ts
```

Это обновит все изображения с SEO-оптимизированными alt-текстами.

### Шаг 3: Проверь результат

```bash
npx tsx scripts/audit-alt-texts.ts
```

Должно показать ~100% изображений с качественными alt-текстами.

### Шаг 4: Деплой

```bash
npm run build
# или твоя команда для деплоя
```

### Шаг 5: Мониторинг (через 2-3 недели)

1. Открой [Google Search Console](https://search.google.com/search-console)
2. Перейди в **Performance** → **Search results**
3. Фильтр: **Search type** → **Image**
4. Смотри рост impressions и clicks

## Формат alt-текстов

### Примеры правильных alt-текстов:

✅ `LEGO Конструктор Железный Человек Marvel купить в Казахстане`
✅ `Barbie Кукла Принцесса Disney вид сбоку`
✅ `Hot Wheels Трек Мертвая петля детальное фото`
✅ `Hasbro Настольная игра Монополия в упаковке`

### Примеры неправильных alt-текстов:

❌ `Изображение товара 1`
❌ `Image`
❌ `фото 1`
❌ `product.jpg`
