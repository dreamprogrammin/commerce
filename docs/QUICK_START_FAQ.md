# Быстрый старт: Автогенерация FAQ

## Что это?

Система автоматически создаёт вопросы-ответы для товаров на основе их данных (возраст, бренд, материал, доставка и т.д.).

## Зачем?

✅ SEO: Google показывает FAQ в результатах поиска
✅ UX: Покупатели сразу видят ответы на популярные вопросы
✅ Экономия времени: не нужно писать вручную для 1000 товаров

## Как использовать?

### Вариант 1: Для одного товара

1. Админка → Товары → Выбрать товар
2. Нажать **"Сгенерировать FAQ"**
3. Готово! Вопросы появятся на странице товара

### Вариант 2: Для всех товаров сразу

1. Админка → Товары (главная страница)
2. Нажать **"Сгенерировать FAQ для всех"**
3. Подтвердить → Дождаться завершения (может занять несколько минут)

## Какие вопросы генерируются?

- ✅ С какого возраста? (если есть min_age_years/max_age_years)
- ✅ Это оригинал? (если есть бренд)
- ✅ Из какого материала? (если есть material_id)
- ✅ Где произведена? (если есть origin_country_id)
- ✅ Как быстро доставите? (всегда)
- ✅ Можно ли вернуть? (всегда)
- ✅ Товар точно есть? (если цена > 20000₸)

## Важно

⚠️ Старые автогенерированные вопросы **перезапишутся** при повторной генерации
✅ Пользовательские вопросы **сохранятся**
