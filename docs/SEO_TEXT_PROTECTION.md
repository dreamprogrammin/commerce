# Защита уникальных SEO-текстов от перезаписи

## Проблема

Кнопка "🤖 Автогенерация" в `/admin/brand-seo` перезаписывала длинные HTML-тексты, написанные вручную контент-менеджером для важных брендов, заменяя их на короткие шаблоны.

## Решение

Добавлена защита уникальных SEO-текстов на уровне базы данных через RPC функцию `safe_upsert_category_brand_seo`.

### Критерии защиты текста

Текст считается **уникальным** (защищённым от перезаписи), если выполняется хотя бы одно условие:

1. **Длина > 300 символов**
2. **Содержит HTML-теги** (кроме простых `<p>`, `<br>`):
   - `<h1>`, `<h2>`, `<h3>`, `<h4>`, `<h5>`, `<h6>`
   - `<div>`, `<section>`, `<article>`
   - `<ul>`, `<ol>`, `<li>`
   - `<strong>`, `<em>`, `<a>`

### Поведение автогенерации

#### Кнопка "🤖 Автогенерация" (безопасный режим)

- **Создаёт** новые записи для комбинаций категория+бренд, где их ещё нет
- **НЕ перезаписывает** уникальные SEO-тексты
- **Обновляет** только шаблонные тексты (короткие, без HTML)
- **Обновляет** `seo_h1`, `seo_title`, `seo_description` даже для защищённых записей

#### Кнопка "🔄 Перезаписать все" (опасный режим)

- **Перезаписывает ВСЕ** записи, включая уникальные тексты
- Требует подтверждения с предупреждением
- Используется только для массового обновления шаблонов

## Технические детали

### RPC функция

```sql
safe_upsert_category_brand_seo(
  p_category_id UUID,
  p_brand_id UUID,
  p_seo_h1 TEXT,
  p_seo_title TEXT,
  p_seo_description TEXT,
  p_seo_text TEXT
) RETURNS JSONB
```

**Возвращает:**

```json
{
  "action": "updated_partial" | "updated_full" | "inserted",
  "message": "Описание действия",
  "protected": true | false
}
```

### Миграция

`supabase/migrations/20260520154300_safe_upsert_category_brand_seo.sql`

### Изменённые файлы

1. **`composables/useSeoTemplates.ts`**
   - Функция `generateSeoForAllCategoryBrands` теперь использует `safe_upsert_category_brand_seo`
   - Режим `overwrite: false` (по умолчанию) — безопасный upsert
   - Режим `overwrite: true` — принудительная перезапись

2. **`types/supabase.ts`**
   - Добавлен тип для новой RPC функции

## Использование

### Для контент-менеджера

1. Напишите уникальный HTML-текст в админке `/admin/brand-seo`
2. Сохраните запись
3. Теперь кнопка "🤖 Автогенерация" **НЕ перезапишет** этот текст
4. Можно безопасно запускать автогенерацию для остальных брендов

### Для разработчика

```typescript
// Безопасная автогенерация (защищает уникальные тексты)
await generateSeoForAllCategoryBrands({ dryRun: false })

// Принудительная перезапись всех текстов
await generateSeoForAllCategoryBrands({ dryRun: false, overwrite: true })
```

## Примеры

### Защищённый текст (НЕ будет перезаписан)

```html
<h2>Конструкторы LEGO для мальчиков в Алматы</h2>
<p>Ухтышка предлагает широкий выбор оригинальных конструкторов LEGO...</p>
<ul>
  <li>Официальная гарантия</li>
  <li>Доставка день-в-день</li>
</ul>
```

### Шаблонный текст (будет обновлён)

```
Ухтышка — это официальный интернет-магазин в Алматы, где можно купить оригинальные конструкторы LEGO.
```
