## 6. 🆕 Полная структура Schema.org индексации

### Иерархия страниц и их schema:

```
📄 Главная (/)
├─ Organization Schema (компания)
├─ Store Schema (магазин)
├─ ItemList (Категории) - топ категорий
├─ ItemList (Brands) - топ-20 брендов
├─ ItemList (Product Lines) - топ-30 линеек
└─ CollectionPage (главная как коллекция)

📄 Категория (/catalog/igrushki-dlya-malchikov)
├─ CollectionPage Schema (категория)
├─ BreadcrumbList Schema
├─ SiteNavigationElement (подкатегории) → Sitelinks
├─ ItemList (Brands) → Бренды в категории
├─ ItemList (Product Lines) → Линейки в категории
├─ ItemList (Materials) → Материалы товаров
├─ ItemList (Countries) → Страны производства
├─ ItemList (Attributes) → Атрибуты (размер, возраст, цвет)
└─ ItemList (Products) → Товары с ценами

📄 Бренд (/brand/lego)
├─ Brand Schema
│  └─ subOrganization (линейки как суб-бренды)
├─ CollectionPage Schema
├─ ItemList (Products) → Товары бренда
└─ BreadcrumbList Schema

📄 Линейка (/brand/lego/star-wars)
├─ Brand Schema
│  └─ parentOrganization (родительский бренд)
├─ CollectionPage Schema
│  └─ mainEntity: ItemList (товары)
└─ BreadcrumbList Schema

📄 Товар (/catalog/products/lego-star-wars-falcon)
├─ Product Schema (полная информация)
├─ Offer Schema (цена, наличие, доставка)
├─ Brand Hierarchy (LEGO → LEGO Star Wars)
├─ AggregateRating (если есть отзывы)
└─ BreadcrumbList Schema
```
