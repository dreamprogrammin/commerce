# 🚀 Быстрый старт: Автоматизация SEO (2 минуты)

## Шаг 1: Применить миграции (30 сек)

```bash
cd /home/malik/projects/commerce
supabase db push
```

## Шаг 2: Открыть админку (10 сек)

Перейти на: `http://localhost:3000/admin/brand-seo`

## Шаг 3: Запустить автогенерацию (1 мин)

1. Нажать кнопку **"🤖 Автогенерация"**
2. Подтвердить действие
3. Дождаться завершения (создаст SEO-тексты для всех комбинаций)

## Шаг 4: Сгенерировать FAQ (30 сек)

1. Нажать кнопку **"❓ Генерация FAQ"**
2. Подтвердить действие
3. Готово!

## ✅ Результат

Теперь все страницы вида `/catalog/{category}?brand={brand}` имеют:
- ✅ Уникальный SEO-текст
- ✅ 4 FAQ с ключевыми запросами
- ✅ Открыты для индексации Google
- ✅ Готовы перехватывать запросы в ChatGPT/Gemini

## 🎯 Проверка

Откройте любую страницу, например:
```
http://localhost:3000/catalog/constructors-root/konstruktory-malchikam?brand=cada
```

Должны увидеть:
- H1: "Конструкторы CADA — купить в Алматы"
- SEO-текст внизу страницы
- FAQ с вопросами про CADA

## 📊 Мониторинг

Через 2-4 недели проверьте:
1. Google Search Console → индексация страниц с `?brand=`
2. ChatGPT → спросите "где купить {ваш_бренд} в Алматы"
