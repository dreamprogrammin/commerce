## 🎯 План улучшений (по приоритету)

### Приоритет 1: Популярные категории в футере (День 1)

**Эффект:** +20-30% к индексации категорий

**Что делать:**

1. Создать composable `usePopularCategories.ts`
2. Добавить блок в `Footer.vue`
3. Получить топ-5 категорий из базы

**Код:**

```typescript
// composables/usePopularCategories.ts
export function usePopularCategories() {
  return useQuery({
    queryKey: ['popular-categories'],
    queryFn: async () => {
      const { data } = await supabase
        .from('categories')
        .select('id, name, slug')
        .eq('is_active', true)
        .order('products_count', { ascending: false })
        .limit(5)
      return data || []
    },
    staleTime: 60 * 60 * 1000, // 1 час
  })
}
```

### Приоритет 2: Популярные товары в футере (День 2)

**Эффект:** +10-15% к индексации товаров

**Что делать:**

1. Создать composable `usePopularProducts.ts`
2. Добавить блок в `Footer.vue`
3. Получить топ-5 товаров по продажам

**Код:**

```typescript
// composables/usePopularProducts.ts
export function usePopularProducts() {
  return useQuery({
    queryKey: ['popular-products'],
    queryFn: async () => {
      const { data } = await supabase
        .from('products')
        .select('id, name, slug, price, final_price')
        .eq('is_active', true)
        .order('sales_count', { ascending: false })
        .limit(5)
      return data || []
    },
    staleTime: 60 * 60 * 1000, // 1 час
  })
}
```
