## 🔴 Критические проблемы (блокируют рост)

### 1. **Отсутствует schema.org разметка для товаров**

**Проблема:** Нет Product schema → Google не понимает, что это товар → нет rich snippets → низкий CTR

**Что делать:**

```vue
<!-- В pages/catalog/products/[slug].vue добавить: -->
<script setup lang="ts">
useSchemaOrg([
  defineProduct({
    name: product.name,
    image: productImages,
    description: cleanDescription(product.description),
    sku: product.sku,
    brand: {
      name: product.brand?.name,
    },
    offers: {
      price: product.price,
      priceCurrency: 'KZT',
      availability: product.stock > 0 ? 'InStock' : 'OutOfStock',
      priceValidUntil: new Date(Date.now() + 365 * 24 * 60 * 60 * 1000).toISOString(),
      url: `https://uhti.kz/catalog/products/${product.slug}`,
    },
    aggregateRating: product.reviews_count > 0
      ? {
          ratingValue: product.average_rating,
          reviewCount: product.reviews_count,
        }
      : undefined,
  }),
])
</script>
```

**Приоритет:** 🔥 Критический
**Срок:** День 1-2
**Эффект:** +30-50% к CTR в выдаче

### 2. **Слабые title и description**

**Проблема:** Позиция 18.24 = страницы на 2-й странице → не кликают

**Текущий формат (плохо):**

```
Title: "Детская кухня | Uhti.kz"
Description: "Купить детскую кухню в интернет-магазине Uhti.kz"
```

**Правильный формат (хорошо):**

```
Title: "Детская кухня деревянная с холодильником — от 15 990 ₸ | Uhti.kz"
Description: "Детская кухня с плитой, холодильником и посудой. Развивает воображение и моторику. ⭐ 4.8 (23 отзыва). Доставка по Алматы за 1 день."
```

**Формула для title:**

```
[Товар] [ключевое свойство] — от [цена] ₸ | [Бренд]
```

**Формула для description:**

```
[Товар] [польза для ребенка]. [Уникальное преимущество]. ⭐ [рейтинг] ([отзывы]). [Доставка].
```

**Приоритет:** 🔥 Критический
**Срок:** День 3-5
**Эффект:** +20-40% к CTR

### 3. **Нет уникальных описаний товаров**

**Проблема:** Если описания копируются с других сайтов → Google считает дублями → не ранжирует

**Что проверить:**

```sql
-- Найти товары без описаний
SELECT id, name, slug, description
FROM products
WHERE description IS NULL
   OR LENGTH(description) < 100
ORDER BY created_at DESC
LIMIT 50;
```

**Шаблон описания для игрушек:**

```markdown
## [Название товара]

**Для кого:** дети от [возраст] лет
**Материал:** [материал] — безопасный, гипоаллергенный
**Размер:** [размеры]

### Что развивает:

- Мелкую моторику и координацию
- Воображение и ролевую игру
- Социальные навыки

### В комплекте:

- [список предметов]

### Почему родители выбирают эту игрушку:

- Прочная конструкция — прослужит годами
- Яркие цвета — привлекают внимание ребенка
- Легко моется

**Гарантия:** 30 дней на возврат
```

**Приоритет:** 🔥 Критический
**Срок:** День 6-15 (по 10-15 товаров в день)
**Эффект:** +50-100% к индексации

## 🟡 Важные проблемы (тормозят рост)

### 4. **Слабая структура категорий**

**Проблема:** Категории не оптимизированы под длинный хвост

**Текущая структура (плохо):**

```
/catalog/kitchen-toys
```

**Правильная структура (хорошо):**

```
/catalog/kitchen-toys
/catalog/kitchen-toys/wooden
/catalog/kitchen-toys/with-refrigerator
/catalog/kitchen-toys/for-girls
/catalog/kitchen-toys/age-3-5
```

**Что делать:**

1. Создать подкатегории по:
   - Материалу (деревянные, пластиковые)
   - Возрасту (3-5 лет, 5-7 лет)
   - Полу (для девочек, для мальчиков)
   - Типу (с холодильником, с плитой, с посудой)

2. Добавить SEO-тексты для каждой подкатегории:

```markdown
# Деревянные детские кухни — экологичные игрушки для ролевых игр

Деревянные кухни для детей от 3 лет — это безопасные и долговечные игрушки...

## Почему деревянные кухни лучше пластиковых:

- Экологичный материал
- Прочнее и долговечнее
- Приятные на ощупь
```

**Приоритет:** 🟡 Важный
**Срок:** День 16-20
**Эффект:** +30-50 новых страниц в индексе

### 5. **Нет внутренней перелинковки**

**Проблема:** Google не понимает, какие страницы важнее

**Что делать:**

3. В футере добавить "Популярные категории"

```vue
<template>

    </div>
  </div>
</template>
```

**Приоритет:** 🟡 Важный
**Срок:** День 21-25
**Эффект:** +20-30% к индексации

### 6. **Нет BreadcrumbList schema**

**Проблема:** Хлебные крошки не отображаются в выдаче

**Что добавить:**

```typescript
// В composables/useBreadcrumbSchema.ts
export function useBreadcrumbSchema(items: IBreadcrumbItem[]) {
  useSchemaOrg([
    defineBreadcrumb({
      itemListElement: items.map((item, index) => ({
        '@type': 'ListItem',
        'position': index + 1,
        'name': item.label,
        'item': item.to ? `https://uhti.kz${item.to}` : undefined,
      })),
    }),
  ])
}
```

**Приоритет:** 🟡 Важный
**Срок:** День 2-3
**Эффект:** +5-10% к CTR

### 8. **Добавить отзывы с schema.org**

**Проблема:** Отзывы есть, но Google их не видит

**Что добавить:**

```typescript
useSchemaOrg([
  defineReview({
    itemReviewed: {
      name: product.name,
    },
    author: {
      name: review.author_name,
    },
    reviewRating: {
      ratingValue: review.rating,
      bestRating: 5,
    },
    reviewBody: review.text,
```
